### Projeto ReactJS + Bootstrap + Json-server API

#### Dependências
* Json-server
* Bootstrap
* Font Awesome
* Axios

#### ReactJS Recursos
* useState
* useEffect

#### como rodar o projeto
* Após baixar do git
* dentro da pasta do projeto executar os passos

* instalando as dependências
    * npm i
* terminal 1 - rodar api
    * npm run api
* terminal 2 - rodar web
    * npm run rdev
